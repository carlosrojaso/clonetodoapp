#Ionic 101

## Paso 1. Instalar IONIC

```bash
$ npm install -g ionic
```

## Paso 2. Crear Proyecto en Blanco.

```bash
$ ionic start -a todo -i us.bootcampwith.todo todo blankls
```

## Paso 3. Agregar Plataformas.

```bash
$ cd todo
$ ionic platform ios
$ ionic platform android
```
## Paso 4. Crear Iconos y SplashScreen.

```bash
$ ionic resources
```
